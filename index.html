Welcome to QuickJobs

QuickJobs is a lightweight job portal demo built for a college lab project. It shows modern HTML5, CSS, JavaScript and PHP integration — all in one simple submission.

Domain: Recruitment / Job Listings

Features Demonstrated

  • HTML5 semantic tags, headings and paragraph formatting.
  • Tables and lists for job data.
  • Forms and validation with JavaScript + PHP (GET & POST).
  • Iframe embed for company location.
  • External, internal and inline CSS; selectors (id, class, group, universal).
  • CSS positioning, floats, responsive layout (basic).
  • JavaScript: variables, operators, loops, functions, cookies, arrays, Date object.
